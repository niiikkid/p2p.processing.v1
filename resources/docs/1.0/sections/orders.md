<a name="description"></a>
## Описание
В данном разделе представлены все сделки совершенные трейдерами/трейдером по его реквизитам. 
Раздел нужен для мониторинга сделок, но также есть возможность досрочно закрыть сделку нажав кнопку "оплачено".
Тогда сделка будет выполнена точно также как если бы она была закрыта автоматически через смс сообщение об переводе средств.

В модально окне (Данные сделки), есть возможность быстро посмотреть все поступившие сообщения трейдеру об преводе средств.

<a name="statuses"></a>
## Статусы
Все сделки имеют всего 3 статуса:
- **Завершен** - сделка была оплачена и успешно исполнена.
- **Отменен** - сделка не была оплачена.
- **Обработка** - ожидание перевода средств от клиента по сделке.

<a name="props"></a>
## Описание параметров
- **Внешний ID** - это id сделки во внешнем сервисе с которого создана сделка.
- **Сумма** - сумма сделки в оригинальной валюте.
- **Курс** - цена покупки 1 USDT.
- **Курс с комиссией** - курс умноженный на % комиссии трейдера.
- **Сумма USDT** - сумма сделки конвертированная в USDT. По формуле: сумма / курс с комиссией.
- **Комиссия %** - комиссия трейдера.
- **Прибыль** - разница между: сумма USDT и сумма / цена конвертации. Или проще говоря это комиссия трейдера.
- **Метод** - платежный метод по которому совершена сделка.
- **Реквизиты** - реквизиты по которым совершена сделка.
- **Коллбек URL** - ссылка по которой отправляются изменения статуса сделки во внешний сервис.
- **Истекает** - время автоматического закрытия сделки в статус "Отменен".
- **Создан** - время созднания сделки.
