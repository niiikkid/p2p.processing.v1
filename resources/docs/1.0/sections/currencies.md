## Валюты
В данном разделе представлены все валюты, по которым доступен прием и обработка платежей. 

Валюты доступные в данный момент: RUB, KZT, UZS. 

Каждая валюта имеет свой обменный курс (продажа/покупка) по отношению к базовой валюте проекта USDT.

<a name="parser"></a>
## Парсер обменных курсов
Для каждой валюты можно настроить парсер цен на обмен валюты. 

Парсер берет цены со страницы ByBit https://www.bybit.com/fiat/trade/otc, а настройки парсера повторяют настройки фильтров на указанной сранице.
