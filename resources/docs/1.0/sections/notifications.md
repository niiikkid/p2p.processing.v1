<a name="description"></a>
## Описание
В данном разделе каждый пользователь может привязать свой аккаунт внутри системы, к своему телеграм аккаунту. 
В таком случае он сможет получать уведомления напрямую в телеграм.

<a name="types"></a>
## Типы уведомлений
- Трейдер:
  - создана новая сделки по реквизиту
  - открыт новый спор по сделке
  - низкий траст баланс (меньше 10% то резерва)
- Мерчант:
  - создана новая сделка в мерчанте
- Также администратор может отправить всем пользователям глобальное уведомление.

<a name="global"></a>
## Глобальное уведомление
Это уведомление которое администратор отправляет всем пользователям. 
В списке уведомлений можно увидеть историю всех отправленных уведомлений за все время, а также
прогресс бар с тем скольким пользователям в данный момент было доставлено уведомление.
