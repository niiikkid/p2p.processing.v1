<a name="description"></a>
## Описание
В данном разделе представлены все доступные платежные системы. Платежная система это своего рода агрегатор реквизитов, 
которые система выдает клиентам для осуществления перевода. Полный список методов может меняться, так как в системе
существует возможность динамического добавления новых платежных методов.

<a name="proprs-description"></a>
## Описание параметров
- **Название** - у каждого платежного метода есть имя. Например: Сбербанк, Альфа-Банк, СБП
- **Код метода** - это технический параметр, который используется для создания платежей через API. Например: sberbank_rub. Где окончание _rub, система добавляет сама в зависимости от выбранной валюты.
- **Вспомогательный метод** - необходим для платежного метода СБП, так как он является своего рода агрегатором платежных методов. C помощью него мы даем инструкцию системе, какие именно платежные методы доступны в рамках метода СБП.
- **Тип реквизитов** - каждый платежный реквизит может быть одного разного типа, например: номер телефона, номер счета, номер банковской карты. Соответственно данный параметр нужен для настройки того какие типы реквизитов доступны для данного платежного метода.
- **Валюта** - настройка валюты в которой будет работать платежный метод.
- **Лимиты операции** - каждые реквизиты в рамках платежного метода имею ограничение на минимальну и максимальную сумму сделки в выбранной валюте.
- **Комиссия трейдера** - это наценка на официальный курс обмена USDT/Валюта метода. Данная наценка является прибылью которую получает трейдер совершая сделки.
- **Время удержания реквизитов** - время того, как долго реквизиты будут выданы на совершение платежа клиенту. Если время истекло, то реквизиты освобождаются для совершения следующей сделки.
- **Метод активен** - параметр включающий и выключающий метод и соответственно все его реквизиты процесса из совершения сделок.

<a name="important"></a>
## Важные моменты
- Каждый платежный метод может быть прикреплен только к одной валюте, поэтому в названии присутствует указание валюты метода. Например: Сбербанк RUB. Если мы когда нибудь захотим добавить в платежный метод "Сбербанк" новую валюту, то мы создадим новый платежный метод для данной валюты. Например: Сбербанк USD
